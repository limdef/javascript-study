# Variable types
- primitive, single item : number, string, boolean, null, undefined, symbol

- object, box container : 여러개의 single item 을 하나로 묶어서 사용 가능

- function, first-class function : function 도 변수에 할당 가능하기 때문에 함수에 인자로 전달 가능, 함수의 리턴 타입으로 가능 


# null, undefined
- null => null로 값이 할당이 되어져 있음
- undefined => 변수가 선언되었지만 아무것도 값이 지정되어 있지 않음

# symbol, create unique identifiers for objects

# Dynamic typing: dynamically typed language
- 할당된 값에 따라 타입이 바뀜
